# Gym Website

## Overview
This project is a simple HTML and CSS webpage for a gym. It serves as an online presence for the gym, providing essential information to potential and current members.

## Project Structure
```
gym-website
├── index.html        # Main entry point of the webpage
├── about.html        # Information about the gym
├── classes.html      # List of classes offered
├── contact.html      # Contact information and form
├── css
│   └── styles.css    # Styles for the webpage
├── js
│   └── main.js       # JavaScript for interactive features
├── assets
│   └── fonts         # Font files for typography
└── README.md         # Documentation for the project
```

## Features
- **Homepage**: Welcoming layout with gym highlights.
- **About Page**: Details about the gym's history, mission, and values.
- **Classes Page**: Comprehensive list of classes with descriptions and schedules.
- **Contact Page**: Easy-to-use contact form for inquiries.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the homepage.
3. Navigate through the other pages using the provided links.
